# CI/CD Pipeline with GitHub Actions

This repository uses GitHub Actions to automate the deployment of the web application.

## Workflow Details

- **Trigger**: On push to the `main` branch.
- **Steps**:
  1. Checkout code.
  2. Set up Node.js.
  3. Install dependencies.
  4. Build the project.
  5. Deploy to GitHub Pages.
